# Raiders Digest - Product Overview

## Purpose
Raiders Digest is a comprehensive companion app for ARC Raiders players, providing essential game information including weapon statistics, loot catalogs, quest tracking, enemy intel, map locations, and trader information. The app serves as a centralized knowledge base for players to reference while playing ARC Raiders.

## Key Features

### Quest Tracking System
- Star important quests for quick access
- Track starred quests directly on the home screen
- Persistent quest state management using AsyncStorage

### Weapons Database
- Complete catalog of all in-game weapons
- Detailed weapon statistics and specifications
- Advanced filtering and search capabilities
- Weapon comparison tools

### Loot Catalog
- Comprehensive loot items database
- Rarity classifications and descriptions
- Loot cheat sheet for quick reference
- Visual item representations

### Enemy Intel
- Information on all ARC threats
- Enemy statistics and behaviors
- Combat strategies and weaknesses

### Map Locations
- Key location markers and descriptions
- Danger zone identification
- Strategic positioning information

### Traders System
- Trader profiles and specialties
- Trading information and inventory details

### Patch Notes
- Version history tracking
- Detailed changelog with categorized updates
- Latest game updates and modifications

### Survival Guide
- Game mechanics explanations
- Tips and strategies for players
- Best practices and recommendations

## Target Users
- ARC Raiders players seeking quick reference information
- New players learning game mechanics
- Experienced players optimizing their gameplay
- Community members tracking game updates

## Value Proposition
- **No External Dependencies**: All data is static and embedded, ensuring fast load times and offline capability
- **Password Protection**: Secure access with device-level authentication persistence
- **Cross-Platform**: Works on web, iOS, and Android through React Native/Expo
- **Responsive Design**: Optimized for both mobile and desktop experiences
- **Cyberpunk Aesthetic**: Immersive terminal-style UI matching ARC Raiders theme

## Access Control
- Password-protected entry (default: `raiders2024`)
- Device authentication persistence via localStorage
- Configurable password system for community management
